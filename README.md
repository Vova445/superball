# Мегабол (Megaball)

Проєкт з архітектурою monorepo.

## Структура проекту

- `frontend/` — клієнтська частина додатку.
- `backend/` — серверна частина додатку.
- `shared/` — спільні ресурси та типи.

## Встановлення та запуск локально

### 1. Бекенд (Python)
- Перейдіть до папки бекенду: `cd backend`
- Створіть віртуальне середовище: `python -m venv venv`
- Активуйте його: `.\venv\Scripts\activate`
- Встановіть залежності: `pip install -r requirements.txt`
- Запустіть сервер: `uvicorn main:app --reload`

### 2. Фронтенд (Next.js)
- Перейдіть до папки фронтенду: `cd frontend`
- Встановіть залежності: `npm install`
- Запустіть у режимі розробки: `npm run dev`

### 3. База даних та Redis
- Оскільки проект більше не використовує Docker, переконайтеся, що **PostgreSQL** та **Redis** встановлені та запущені на вашій системі локально.
- Перевірте налаштування у файлі `.env`.
